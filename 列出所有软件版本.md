## 因为各种软件版本的不兼容，造就了无尽的痛苦，因此有必要记录所有软件版本的安装和使用
### pip，pip3列出所有软件的版本
```
# pip install 软件名==
# pip3 install 软件名==
pip3 install sshuttle==
pip3 install sshuttle==0.78.5
```

### npm查看插件版本列表
```
# npm查看插件版本列表
npm view compression-webpack-plugin versions
# npm安装指定版本的插件
npm install compression-webpack-plugin@6.0.2 --save-dev --verbose

```
